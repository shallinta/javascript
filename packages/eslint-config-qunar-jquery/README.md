# eslint-config-qunar-jquery

Qunar's ESLint config, following our styleguide.

## 安装
```
npm i eslint-config-qunar-jquery
```

## 使用

在 eslint 配置文件中继承 `eslint-config-qunar-jquery` ：

```js
// .eslintrc.js
module.exports = {
  extends: ['eslint-config-qunar-jquery']
};
```

```json
// .eslintrc
{
  "extends": ["eslint-config-qunar-jquery"]
}
```
