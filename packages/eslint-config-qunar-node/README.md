# eslint-config-qunar-node

Qunar's ESLint config, following our styleguide.

## 安装
```
npm i eslint-config-qunar-node
```

## 使用

在 eslint 配置文件中继承 `eslint-config-qunar-node` ：

```js
// .eslintrc.js
module.exports = {
  extends: ['eslint-config-qunar-node']
};
```

```json
// .eslintrc
{
  "extends": ["eslint-config-qunar-node"]
}
```
